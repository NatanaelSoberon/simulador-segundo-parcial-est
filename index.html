Unidad 3 · Pruebas de Hipótesis (una muestra)

Parámetro vs estadístico: las hipótesis se formulan sobre parámetros poblacionales (μ, σ, p), no sobre lo que medimos en la muestra (x̄, s, p̂).

H₀ vs H₁: H₀ es el "statu quo" o "no hay efecto". H₁ es lo que queremos probar.

  • Bilateral: palabra clave "diferente". H₁: μ ≠ μ₀
  • Unilateral derecha: "aumenta / mayor / supera". H₁: μ > μ₀
  • Unilateral izquierda: "disminuye / menor / reduce". H₁: μ < μ₀
  • H₀ siempre incluye la igualdad (=, ≥, ≤)

Errores:

  • Error Tipo I (α): rechazar H₀ siendo verdadera (falso positivo). "Creer que hay efecto cuando no lo hay".
  • Error Tipo II (β): no rechazar H₀ siendo falsa (falso negativo). "No detectar un efecto que sí existe" → costo de oportunidad.

Z vs t:

  • Z: cuando σ poblacional es conocida (raro en marketing).
  • t: cuando σ es desconocida y se estima con s muestral (el caso típico). gl = n − 1.

Regla del p-valor:

Si p-valor < α → Rechazo H₀ (hay evidencia significativa).
Si p-valor ≥ α → No rechazo H₀ (no hay evidencia suficiente).

Unidad 4 · Comparaciones avanzadas y modelado

1. Diferencia de medias (dos muestras):

  • Independientes: grupos distintos (A/B con usuarios diferentes). Prueba t de Welch si σ desconocidas.
  • Apareadas: mismos sujetos en dos momentos (antes/después) o pares emparejados. Se analizan las diferencias dᵢ.
  • Estadístico t (independientes): t = (x̄₁ − x̄₂) / √(s₁²/n₁ + s₂²/n₂)

2. ANOVA de un factor (≥ 3 grupos):

  • Evita la inflación del Error Tipo I que ocurre al hacer múltiples pruebas t.
  • H₀: μ₁ = μ₂ = … = μₖ    H₁: al menos una media es diferente.
  • Estadístico F = MSB / MSW (variabilidad entre grupos ÷ dentro de grupos).
  • Supuestos: independencia, normalidad, homocedasticidad.
  • Si rechazás H₀ → aplicar post-hoc (Tukey, Bonferroni) para identificar qué pares difieren.

3. Chi-Cuadrado (χ²):

  • Bondad de ajuste: 1 variable categórica vs distribución esperada.
  • Independencia: 2 variables categóricas (tabla de contingencia).
  • Estadístico: χ² = Σ (O − E)² / E
  • Supuesto crítico: frecuencias esperadas ≥ 5 (Cochran). Si no se cumple, agrupar categorías o usar Fisher exacto.

4. Correlación de Pearson (r):

  • Rango: −1 a +1. Mide fuerza y dirección de relación lineal entre dos variables cuantitativas.
  • Siempre acompañar con diagrama de dispersión (Pearson no detecta no-linealidades).
  • Correlación NO implica causalidad (variable oculta, causalidad inversa, espuria).

5. Regresión lineal simple:

  • Modelo: Ŷ = b₀ + b₁·X
  • Pendiente b₁: cambio promedio en Y por cada unidad de X. Es el coeficiente accionable.
  • Intercepto b₀: valor predicho de Y cuando X=0 (cuidado con extrapolación).
  • R²: proporción de varianza de Y explicada por X (entre 0 y 1). En regresión simple, R² = r².
